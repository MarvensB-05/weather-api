# 🌦️ Weather App

## 🚀 Overview
A responsive web application that displays real-time weather data using the OpenWeatherMap API. Users can search any city and instantly view live weather conditions.

---

## ✨ Features
- Real-time weather data
- City search functionality
- Temperature, humidity, wind speed display
- Error handling for invalid cities
- Clean responsive UI

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

---

## 🧠 Concepts Used
- REST API integration
- Asynchronous JavaScript (fetch / async-await)
- DOM manipulation
- Event handling
- Error handling

---

## 📂 How to Run
1. Get a free API key from OpenWeatherMap  
2. Insert API key into `script.js`  
3. Open `index.html` in browser
